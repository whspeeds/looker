Jobdesk IT – Website, Hosting, Domain, Server, & PBN Management

1. Manajemen Website & CMS

  • Mengelola dan memelihara CMS (seperti WordPress, Joomla, atau custom CMS).
  • Memastikan website berjalan optimal dan bebas error (error 500, 404, dll).
  • Update tema, plugin, dan patch keamanan secara berkala.
  • Optimasi kecepatan dan performa website (minify, lazyload, dll).

2. Hosting dan Server

  • Setup dan konfigurasi shared hosting, VPS, atau dedicated server.
  • Monitoring uptime server dan respon time.
  • Backup otomatis mingguan/bulanan dan restore jika dibutuhkan.
  • Install panel (DirectAdmin, cPanel, Webmin, dll) dan software pendukung.

3. Domain Management

  • Mengelola pembelian, perpanjangan, dan migrasi domain.
  • Setup DNS dan konfigurasi record (A, MX, TXT, CNAME, dll).
  • Integrasi domain dengan Cloudflare dan layanan pihak ketiga lainnya.

4. Cloudflare dan Keamanan

  • Setup Cloudflare untuk proteksi DDoS dan caching statis.
  • Setup SSL (baik dari Cloudflare maupun Let's Encrypt/manual).
  • Menangani redirect, firewall rules, dan keamanan dasar di Cloudflare.

5. PBN Management

  • Setup dan maintain jaringan PBN (Private Blog Network).
  • Mengelola IP berbeda, footprints, dan strategi keamanan.
  • Deploy konten secara manual atau otomatis ke PBN.
  • Mengelola expired domain, pengecekan DA/PA/Spam Score, dll.

6. SEO (Nilai Tambah)

  • Memahami dasar-dasar SEO On-Page dan teknikal (struktur URL, meta tag, sitemap, robots.txt, dll).
  • Kolaborasi dengan tim konten untuk memastikan struktur website SEO friendly.
  • Audit teknikal SEO dasar menggunakan Google Search Console atau tools lainnya.

7. HTML & Static File Handling

  • Bisa mengedit HTML dan file statis (CSS, JS, gambar).
  • Bisa melakukan upload file via FTP, File Manager, atau SSH.
  • Optimasi gambar dan file statis untuk kecepatan (WebP, minify, gzip).

8. ChatGPT dan Otomatisasi

  • Menggunakan ChatGPT untuk membantu generate konten, coding ringan, atau debug script.
  • Otomatisasi tugas-tugas sederhana menggunakan tools seperti Zapier atau cron job.

Tools yang Wajib Dikuasai

  • FTP Client (FileZilla), SSH
  • Cloudflare, DNS Management
  • DirectAdmin / cPanel / VPS Management
  • WordPress dan CMS lainnya
  • SEO Tools (Google Search Console, Ahrefs)
  • ChatGPT / AI tools untuk efisiensi kerja
  • Google Drive/Docs untuk dokumentasi teknis
